AURVANDIL

About

Aurvandil is a database software to store and access the star data from the GAIA archive. The current version of the software is built to handle the Gaia Data Release 3 (GDR3). Current capabilities of the software allow Aurvandil to build HEALPix sphere databses within PostgreSQL, add stars to that database from the GAIA GDR3 source files, and cone search for stars with filters for any of the parameters listed in the GDR3 files.
