Adding Files to the database

Overview

The GAIA Data Release 3 (GDR3) stores data for public use in comma seperated value (CSV) files. The Aurvandil software reads these files, determines the HEALPix pixel each star lies in, then adds it to the respective pixel database.

Technical Explanation

The Aurvandil software makes use of the HEALPix Java library to organize the stars into pixels of equal area on the celestial sphere. The HEALPix library comes with functionality to convert a set of coordinates to a location on the HEALPix sphere that correspondes to a a specific pixel. The GAIA data release 3 (gdr3) stores the right ascension in degrees in a range of 0 to 360 degrees and the declination in degrees from -90 to 90 degrees. These values must be converted to HEALPix coordinates that range from 0 to Pi for latitude and longitude. The star data is then added as a row to the pixel's corresponding PostgreSQL database.
